## နိဒါန်း

မြန်မာစကားလုံး ခွဲခြမ်းစိတ်ဖြာမှု တည်းဖြတ်ကိရိယာသည် မြန်မာဘာသာစကားအတွက် အရည်အသွေးမြင့် စကားလုံးခွဲခြမ်းမှု ဒေတာအစုများ ဖန်တီးရာတွင် အထောက်အကူပြုရန် ဒီဇိုင်းထုတ်ထားသော အထူးကိရိယာတစ်ခုဖြစ်ပါသည်။ ဤတည်းဖြတ်ကိရိယာသည် စာသားတွင် စကားလုံးနယ်နိမိတ်များကို မှတ်သားရာတွင် သင့်ဒေတာအစုတစ်လျှောက် တသမတ်တည်း ထိန်းသိမ်းထားနိုင်စေမည့် နားလည်လွယ်သော အင်တာဖေ့စ်တစ်ခုကို ပံ့ပိုးပေးပါသည်။

### အဓိကအင်္ဂါရပ်များ
- မြန်မာစာသား တိကျစွာ စီမံဆောင်ရွက်ရန် **ဝဏ္ဏအခြေခံ ခွဲခြမ်းမှု**
- တသမတ်တည်းဖြစ်အောင် **အချိန်နှင့်တပြေးညီ ပဋိပက္ခရှာဖွေတွေ့ရှိခြင်း**
- **နှစ်မျိုးသုံး စနစ်** (ခွဲခြမ်းမှု စနစ်နှင့် တည်းဖြတ်စနစ်)
- အသုံးများသော ပုံစံများအတွက် **ပါးနပ်သောအော်တို ပြင်ဆင်မှု**
- **နှစ်ဘာသာစကား အင်တာဖေ့စ်** (အင်္ဂလိပ်နှင့် မြန်မာ)
- **ဒေတာသန့်ရှင်းရေး ကိရိယာများ** အတွင်းထည့်သွင်းထားခြင်း

## စတင်အသုံးပြုခြင်း

### ထည့်သွင်းခြင်းနှင့် တပ်ဆင်ခြင်း

၁။ **မှီခိုမှုများကို ထည့်သွင်းပါ**
   ```bash
   npm install
   ```

၂။ **ဖွံ့ဖြိုးရန်ဆာဗာကို လည်ပတ်ပါ**
   ```bash
   npm run dev
   ```

၃။ **ဘရောက်ဇာတွင် ဖွင့်ပါ**
   - [http://localhost:3000](http://localhost:3000) သို့ သွားပါ

### ပထမဆုံးစာသားကို တင်ခြင်း

၁။ ကိရိယာတန်းရှိ **Import** ခလုတ်ကို နှိပ်ပါ
၂။ မြန်မာစာသား ပါဝင်သော စာသားဖိုင်ကို ရွေးပါ
၃။ တည်းဖြတ်ကိရိယာသည် အစပိုင်း ဝဏ္ဏခွဲခြမ်းမှုကို အလိုအလျောက် လုပ်ဆောင်မည်
၄။ အပိုင်းများကို ပြန်လည်သုံးသပ်၍ ပြင်ဆင်ခြင်း စတင်ပါ

## အခြေခံသဘောတရားများ

### ဝဏ္ဏများနှင့် စကားလုံးများ

တည်းဖြတ်ကိရိယာသည် **ဝဏ္ဏအဆင့်** ကို အခြေခံယူနစ်အဖြစ် လုပ်ဆောင်ပါသည်:
- **ဝဏ္ဏ**: အသေးဆုံး အသံထွက်နိုင်သော ယူနစ် (ဥပမာ - `မြန်`, `မာ`)
- **စကားလုံး**: တစ်ခု သို့မဟုတ် တစ်ခုထက်ပိုသော ဝဏ္ဏများ ပေါင်းစပ်ထားခြင်း (ဥပမာ - `မြန်မာ`)

### အပိုင်းများနှင့် ဘလောက်များ

စာသားကို **အပြန်အလှန်လုပ်ဆောင်နိုင်သော ဘလောက်များ** အဖြစ် ပြသထားပါသည်:
- ဘလောက်တစ်ခုစီသည် စကားလုံးအပိုင်းတစ်ခုကို ကိုယ်စားပြုသည်
- လွယ်ကူစွာ သွားလာရန် ဘလောက်များကို မြင်သာအောင် ခွဲထားသည်
- လက်ရှိလုပ်ဆောင်နေသော ဘလောက်တွင် အပြာရောင်နယ်နိမိတ် ပြသသည်

### ပဋိပက္ခရှာဖွေတွေ့ရှိခြင်း

စကားလုံးတစ်လုံးတည်းကို မတူကွဲပြားစွာ ခွဲခြမ်းထားသည့်အခါ စနစ်သည် အလိုအလျောက် ရှာဖွေတွေ့ရှိပါသည်:
- ပဋိပက္ခဥပမာ: `မြန် မာ` နှင့် `မြန်မာ`
- ပဋိပက္ခများကို အဝါရောင်ဖြင့် မီးမောင်းထိုးပြသသည်
- ဘေးတန်းတွင် ပဋိပက္ခသတိပေးချက်များ ပြသသည်

## တည်းဖြတ်စနစ်များ

### ခွဲခြမ်းမှုစနစ် (မူလစနစ်)

ဤစနစ်သည် စကားလုံးနယ်နိမိတ်များ ချိန်ညှိရာတွင် သင်၏ အဓိကအလုပ်လုပ်ရသော စနစ်ဖြစ်ပါသည်။

**သွားလာခြင်း:**
- အပိုင်းများကြားတွင် သွားလာရန် **မြှားသော့များ** (← →) ကို အသုံးပြုပါ
- ကာဆာရှိသောနေရာတွင် အပိုင်းကို ခွဲရန် **Space** ကို အသုံးပြုပါ
- အရင်အပိုင်းနှင့် ပေါင်းရန် **Backspace** ကို အသုံးပြုပါ

**မြင်သာသော အညွှန်းများ:**
- အပြာနယ်နိမိတ်: လက်ရှိ တက်ကြွသော အပိုင်း
- အဝါနောက်ခံ: ပဋိပက္ခရှိသော အပိုင်း
- ပုံမှန်နောက်ခံ: သာမန်အပိုင်း

### တည်းဖြတ်စနစ်

စာလုံးပေါင်းအမှားများ ပြင်ဆင်ရန် သို့မဟုတ် အက္ခရာအဆင့် စာသားကို ပြုပြင်ရန် ဤစနစ်ကို အသုံးပြုပါ။

**တည်းဖြတ်စနစ်သို့ ဝင်ရောက်ခြင်း:**
- မည်သည့်အပိုင်းတွင်မဆို နှစ်ကြိမ်နှိပ်ပါ
- သို့မဟုတ် အပိုင်းပေါ်တွင်ရှိစဉ် **F2** ကို နှိပ်ပါ

**တည်းဖြတ်စနစ်တွင်:**
- ကာဆာသည် အက္ခရာချင်း သွားလာသည် (သာမန်စာသားတည်းဖြတ်ကိရိယာကဲ့သို့)
- လိုအပ်သလို စာသားကို တည်းဖြတ်ပါ
- သိမ်းဆည်း၍ ခွဲခြမ်းမှုစနစ်သို့ ပြန်ရန် **Enter** ကို နှိပ်ပါ
- သိမ်းဆည်းခြင်းမပြုဘဲ ပြန်ရန် **Esc** ကို နှိပ်ပါ

**အရေးကြီးချက်:** တည်းဖြတ်မှုများ သိမ်းဆည်းပြီးနောက်၊ စာသားကို ဝဏ္ဏများအဖြစ် အလိုအလျောက် ပြန်လည်ခွဲခြမ်းမည်။

## အပိုင်းများနှင့် အလုပ်လုပ်ခြင်း

### အပိုင်းခွဲခြင်း

**ဥပမာ:** `မြန်မာနိုင်ငံ` ကို `မြန်မာ` နှင့် `နိုင်ငံ` အဖြစ် ခွဲခြင်း

၁။ `မြန်မာနိုင်ငံ` အပိုင်းကို နှိပ်ပါ
၂။ `မြန်မာ` နောက်တွင် ကာဆာကို နေရာချရန် မြှားသော့များကို အသုံးပြုပါ
၃။ **Space** ကို နှိပ်ပါ
၄။ ရလဒ်: `[မြန်မာ] [နိုင်ငံ]`

### အပိုင်းများ ပေါင်းခြင်း

**ဥပမာ:** `မြန် မာ` ကို `မြန်မာ` အဖြစ် ပေါင်းခြင်း

၁။ ဒုတိယအပိုင်း `မာ` ကို နှိပ်ပါ
၂။ **Backspace** ကို နှိပ်ပါ
၃။ ရလဒ်: `[မြန်မာ]`

### အစုလိုက်လုပ်ဆောင်ချက်များ

အပိုင်းတစ်ခုကို ပြုပြင်သောအခါ၊ စနစ်သည် ပြောင်းလဲမှုများကို ကမ္ဘာလုံးဆိုင်ရာ လုပ်ဆောင်နိုင်သည်:
၁။ သင့်ပြောင်းလဲမှုကို ပြုလုပ်ပါ (ခွဲခြင်း သို့မဟုတ် ပေါင်းခြင်း)
၂။ ပဋိပက္ခများ တွေ့ရှိပါက၊ ဒိုင်ယာလော့ ပေါ်လာမည်
၃။ ရွေးချယ်ပါ:
   - **အားလုံးပြင်ပါ**: စာရွက်စာတမ်းအတွင်း ပေါ်လာသမျှအားလုံးတွင် လုပ်ဆောင်ပါ
   - **ဤကိစ္စတစ်ခုတည်းအတွက်**: ပြောင်းလဲမှုကို ဒေသန္တရ ထားရှိပါ
   - **မလုပ်ဆောင်ပါ**: ပြောင်းလဲမှုကို နောက်ပြန်ဆုတ်ပါ

## ပဋိပက္ခ ရှာဖွေတွေ့ရှိခြင်းနှင့် ဖြေရှင်းခြင်း

### ပဋိပက္ခများကို နားလည်ခြင်း

စာသားတစ်ခုတည်း မတူညီသော ခွဲခြမ်းမှုဖြင့် ပေါ်လာသောအခါ ပဋိပက္ခ ဖြစ်ပေါ်သည်:

**ဥပမာ:**
- စာကြောင်း ၅: `ကျောင်း သား` (ခွဲထား)
- စာကြောင်း ၄၂: `ကျောင်းသား` (ပေါင်းထား)

### ပဋိပက္ခဒိုင်ယာလော့ ရွေးချယ်မှုများ

ပဋိပက္ခကို ဖြစ်ပေါ်စေသောအခါ၊ သင့်အား ဤရွေးချယ်မှုများပါသော ဒိုင်ယာလော့ တစ်ခု မြင်ရမည်:

| ရွေးချယ်မှု | မည်သည့်အခါ အသုံးပြုရမည် | သက်ရောက်မှု |
|--------|-------------|--------|
| **အားလုံးပြင်ပါ** | ခွဲခြမ်းမှုသစ်သည် အားလုံးအတွက် မှန်ကန်သည် | စာရွက်စာတမ်းအတွင်း ပေါ်လာသမျှအားလုံးကို အဆင့်မြှင့်တင်သည် |
| **ဤကိစ္စတစ်ခုတည်း** | မတူညီသော အကြောင်းအရာများတွင် မတူညီသော ခွဲခြမ်းမှု လိုအပ်သည် | ဤစကားလုံးကို အကြောင်းအရာပေါ်မူတည်သည်ဟု မှတ်သားသည် |
| **မလုပ်ဆောင်ပါ** | သင်အမှား လုပ်မိသည် | သင့်ပြောင်းလဲမှုကို ပြန်ဆုတ်သည် |

### ပဋိပက္ខဘေးတန်း

ညာဘက်တန်းတွင် ပြသသည်:
- သင့်စာရွက်စာတမ်းအတွင်း **တက်ကြွသော ပဋိပက္ခများ**
- သင့်ယခင်ရွေးချယ်မှုများအပေါ် အခြေခံ၍ **အကြံပြုထားသော ပြင်ဆင်မှုများ**
- မှတ်သားခြင်း တိုးတက်မှုဆိုင်ရာ **စာရင်းအင်းများ**

---

## လျင်မြန်သော့ချက်များ

### ကမ္ဘာလုံးဆိုင်ရာ လျင်မြန်သော့များ

| လျင်မြန်သော့ | လုပ်ဆောင်ချက် |
|----------|--------|
| `Ctrl + S` | တိုးတက်မှုကို သိမ်းဆည်းပါ |
| `Ctrl + Z` | နောက်ဆုံးလုပ်ဆောင်ချက်ကို နောက်ပြန်ဆုတ်ပါ |
| `Ctrl + Shift + Z` | ပြန်လည်လုပ်ဆောင်ပါ |
| `Ctrl + F` | စာသားရှာပါ |

### ခွဲခြမ်းမှုစနစ်

| လျင်မြန်သော့ | လုပ်ဆောင်ချက် |
|----------|--------|
| `←` `→` | အပိုင်းများကြား သွားလာပါ |
| `Space` | ကာဆာတွင် အပိုင်းခွဲပါ |
| `Backspace` | အရင်အပိုင်းနှင့် ပေါင်းပါ |
| `F2` | တည်းဖြတ်စနစ်သို့ ဝင်ပါ |
| `Delete` | လက်ရှိအပိုင်းကို ဖျက်ပါ |

### တည်းဖြတ်စနစ်

| လျင်မြန်သော့ | လုပ်ဆောင်ချက် |
|----------|--------|
| `Enter` | သိမ်းဆည်း၍ တည်းဖြတ်စနစ်မှ ထွက်ပါ |
| `Esc` | မလုပ်ဆောင်ဘဲ တည်းဖြတ်စနစ်မှ ထွက်ပါ |
| `←` `→` | ကာဆာကို အက္ခရာချင်း ရွှေ့ပါ |

## ကိရိယာတန်းလျှောက်လှမ်းများ

### ဖိုင်လုပ်ဆောင်ချက်များ

- **သွင်းယူပါ**: ကုန်ကြမ်းမြန်မာစာသားဖိုင်များကို တင်ပါ
- **ထုတ်ယူပါ**: ခွဲခြမ်းထားသော ဒေတာအစုကို ဒေါင်းလုဒ်လုပ်ပါ (JSON ဖော်မတ်)
- **သိမ်းဆည်းပါ**: လက်ရှိတိုးတက်မှုကို သိမ်းဆည်းပါ

### သန့်ရှင်းရေးကိရိယာများ

- **စာသားရှင်းပါ**: မမြင်နိုင်သော အက္ခရာများကို ဖယ်ရှားပါ (ZWSP, ZWNJ)
- **နေရာလပ်များကို ပုံမှန်လုပ်ပါ**: များစွာသော နေရာလပ်များကို တစ်ခုတည်းနှင့် အစားထိုးပါ
- **အော်တို-ခွဲခြမ်းပါ**: အလိုအလျောက် ခွဲခြမ်းမှုကို ပြန်လည်လုပ်ဆောင်ပါ

### ကြည့်ရှုမှုရွေးချယ်မှုများ

- **ပဋိပက္ခများပြပါ**: ပဋိပက္ခမီးမောင်းထိုးပြခြင်းကို ဖွင့်/ပိတ်ပါ
- **စာရင်းအင်းများ**: မှတ်သားခြင်း တိုးတက်မှုကို ကြည့်ပါ
- **ဘာသာစကား**: အင်္ဂလိပ်နှင့် မြန်မာ UI ကြား ပြောင်းပါ

### ဆက်တင်များ

- **ခွဲခြမ်းမှုစည်းမျဉ်းများ**: အသေးစိတ်အဆင့်ကို စီစဉ်ပါ (စကားလုံး/စကားစု/ဝဏ္ဏ အဆင့်)
- **အော်တို-သိမ်းဆည်းမှု**: အလိုအလျောက် သိမ်းဆည်းခြင်းကို ဖွင့်/ပိတ်ပါ
- **အပြင်အဆင်**: အလင်းနှင့် အမှောင် စနစ်များကြား ပြောင်းပါ

---

## အကောင်းဆုံးအလေ့အကျင့်များ

### စတင်မီတွင်

၁။ သန့်ရှင်းရေးကိရိယာများကို အသုံးပြု၍ **သင့်စာသားကို အရင်သန့်ရှင်းပါ**
၂။ **အော်တို-ခွဲခြမ်းမှုကို ပြန်လည်သုံးသပ်ပါ** - များသောအားဖြင့် ၈၀-၉၀% တိကျသည်
၃။ **အစမှ စတင်၍** စနစ်တကျ လုပ်ဆောင်ပါ

### မှတ်သားနေစဉ်အတွင်း

၁။ လျင်မြန်သော လုပ်ဆောင်မှုအတွက် **လျင်မြန်သော့ချက်များကို အသုံးပြုပါ**
၂။ ပေါ်လာသောအခါ **ပဋိပက္ခများကို ချက်ခြင်းပြင်ပါ**
၃။ သင့်ခွဲခြမ်းမှုဆုံးဖြတ်ချက်များနှင့် **တသမတ်တည်း ဖြစ်ပါစေ**
၄။ **မကြာခဏ သိမ်းဆည်းပါ** (သို့မဟုတ် အော်တို-သိမ်းဆည်းမှု ဖွင့်ပါ)

### မရေရာသော ကိစ္စများကို ကိုင်တွယ်ခြင်း

အကြောင်းအရာပေါ် မူတည်၍ စကားလုံးအချို့ကို မတူညီစွာ ခွဲခြမ်းနိုင်သည်:

**ဥပမာ:** `ပြောကြောင်း`
- ကြိယာစကားစုအဖြစ်: `ပြော ကြောင်း` (ပြောခဲ့သည်မှာ)
- နာမ်အဖြစ်: `ပြောကြောင်း` (ပြောခဲ့သည့်အရာ)

**ဖြေရှင်းချက်:** အကြောင်းအရာပေါ်မူတည်သော စကားလုံးများကို မှတ်သားရန် ပဋိပက္ခဒိုင်ယာလော့တွင် "ဤကိစ္စတစ်ခုတည်း" ကို အသုံးပြုပါ။

### အရည်အသွေးထိန်းချုပ်မှု

ခဏခဏတိုင်း စစ်ဆေးပါ:
- **ပဋိပက္ခအရေအတွက်** - တက်ကြွသော ပဋိပက္ခ သုညအတွက် ရည်မှန်းပါ
- **မှတ်သားခြင်း တိုးတက်မှု** - ဘေးတန်းတွင် ပြသထားသည်
- **တသမတ်တည်းဖြစ်မှု** - သင့်အစောပိုင်းအလုပ်ကို ရံဖန်ရံခါ ပြန်လည်သုံးသပ်ပါ

---

## ပြဿနာဖြေရှင်းခြင်း

### အသုံးများသော ပြဿနာများ

**မေး: ဖိုင်ကြီးများနှင့် တည်းဖြတ်ကိရိယာ နှေးကွေးသည်**
- ဖြေ: ဆက်တင်များတွင် virtualized rendering ရွေးချယ်မှုကို အသုံးပြုပါ
- ဖိုင်ကြီးများ (>၅၀၀၀ စာကြောင်းများ) တင်ရန် အချိန်ယူနိုင်သည်

**မေး: မှားယွင်းသော ရွေးချယ်မှုပေါ်တွင် "အားလုံးပြင်ပါ" ကို မတော်တဆရွေးမိသည်**
- ဖြေ: ချက်ခြင်း `Ctrl + Z` ကို နှိပ်၍ နောက်ပြန်ဆုတ်ပါ
- ဘေးတန်းရှိ နောက်ပြန်ဆုတ်ခြင်း မှတ်တမ်းကို စစ်ဆေးပါ

**မေး: သွင်းပြီးနောက် စာသား ဖတ်မရပါ**
- ဖြေ: သင့်ဖိုင်သည် UTF-8 encoded ဖြစ်ကြောင်း သေချာပါစေ
- ကိရိယာတန်းရှိ "စာသားရှင်းပါ" လုပ်ဆောင်ချက်ကို စမ်းကြည့်ပါ

**မေး: သင်မျှော်လင့်သော နေရာတွင် အပိုင်းများ မခွဲပါ**
- ဖြေ: တည်းဖြတ်ကိရိယာသည် ဝဏ္ဏနယ်နိမိတ်များကို အသုံးပြုသည်
- လိုအပ်ပါက အောက်ခံစာသားကို ပြုပြင်ရန် တည်းဖြတ်စနစ်ကို အသုံးပြုပါ

**မေး: မြန်မာစာသားတွင် အင်္ဂလိပ်စကားလုံးများကို မည်ကဲ့သို့ ကိုင်တွယ်မည်နည်း?**
- ဖြေ: အင်္ဂလိပ်စကားလုံးများကို တစ်ခုတည်းသော အပိုင်းများအဖြစ် ထားရှိသည်
- လိုအပ်ပါက တည်းဖြတ်စနစ်တွင် ၎င်းတို့ကို အမှတ်အသားဖြင့် ခွဲနိုင်သည်

### အကူအညီရယူခြင်း

- အသေးစိတ်ဗိမာန်လက်ရာအတွက် ဒီဇိုင်းသတ်မှတ်ချက်စာရွက်စာတမ်းများကို စစ်ဆေးပါ
- FAQ အပိုင်းကို ပြန်လည်သုံးသပ်ပါ (ကာလအတွင်း လာမည်)
- ချွတ်ယွင်းချက်များကို GitHub Issues မှတစ်ဆင့် အစီရင်ခံပါ

## ဒေတာဖော်မတ်

### ထုတ်ယူမှုဖော်မတ်

ခွဲခြမ်းထားသော ဒေတာကို JSON အဖြစ် ထုတ်ယူသည်:

```json
{
  "project_meta": {
    "name": "ကျွန်ုပ်၏_ခွဲခြမ်းမှု_ပရောဂျက်",
    "created_at": "2026-02-10",
    "total_lines": 150
  },
  "content": [
    {
      "id": 1,
      "original_text": "မြန်မာနိုင်ငံသားများ",
      "segments": ["မြန်မာ", "နိုင်ငံ", "သား", "များ"],
      "status": "reviewed"
    }
  ]
}
```

### သွင်းယူမှုလိုအပ်ချက်များ

- **ဖော်မတ်**: ရိုးရှင်းသောစာသား (.txt) ဖိုင်များ
- **ကုဒ်ရေးသားမှု**: UTF-8
- **အကြောင်းအရာ**: မြန်မာယူနီကုဒ်စာသား
- **ဖွဲ့စည်းပုံ**: တစ်ကြောင်းလျှင် စာကြောင်းတစ်ကြောင်း (အကြံပြုထားသော)

## နောက်ထပ် အဆင့်များ

အခြေခံများနှင့် သက်တောင့်သက်သာဖြစ်သောအခါ:

၁။ ခွဲခြမ်းမှုစည်းမျဉ်းများကို စိတ်ကြိုက်ပြင်ဆင်ရန် **အသေးစိတ်အဆင့်ဆက်တင်များ** ကို စူးစမ်းပါ
၂။ ပေါင်းစပ်စကားလုံးများအတွက် **N-gram အကြံပြုစနစ်** ကို စမ်းကြည့်ပါ
၃။ နည်းပညာဆိုင်ရာအသုံးအနှုန်းများအတွက် **ပရောဂျက်သီးခြား အဘိဓာန်များ** ကို တပ်ဆင်ပါ
၄။ သင့် NLP pipeline နှင့် ပေါင်းစည်းရန် **ထုတ်ယူမှုရွေးချယ်မှုများ** ကို အသုံးပြုပါ

**မှတ်သားခြင်း ပျော်ရွှင်ပါစေ! 🎉**
